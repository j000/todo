## Backend:

- Node.js
- Express
- Sequelize + sqlite3

### API

| Method | Path       | Description            |
|--------|------------|------------------------|
| GET    | /todos     | Get all todos items    |
| GET    | /todos/:id | Get one todo item      |
| POST   | /todos     | Create a new todo item |
| PUT    | /todos/:id | Update a todo item     |
| DELETE | /todos/:id | Delete a new todo item |
